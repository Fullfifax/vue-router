## Usage
Step 1: Clone this repository 
```
git clone https://github.com/Fullfifax/vue-router.git
```
Step 2: Browse to your local repository
```
cd vue-router
```
Step 3: Install dependencies
```
yarn install
```
### Compiles and hot-reloads for development
```
yarn serve
```
